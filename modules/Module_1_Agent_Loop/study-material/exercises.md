# Module 01 — Exercises

## Exercise 1 — Spot the level
**Goal:** Classify three real tasks into the four architecture levels and justify each.
**Steps:**
1. Take three tasks the learner cares about (work or personal).
2. For each, decide L1–L4 and name the deciding factor (branching, autonomy, tool choice).
3. Identify one task that would force a jump to the next level, and why.
**Done when:** Each task has a level + a one-line justification.

## Exercise 2 — Trace the loop
**Goal:** Hand-trace the agent loop for a single query (e.g. "what's the weather and should I
bring an umbrella?").
**Steps:**
1. Write the prompt/context.
2. Write the intent the LLM would form.
3. Name the tool(s) called and the observation returned.
4. Decide whether the loop repeats or stops, and why.
**Done when:** The learner can point to where stop_reason fires.
