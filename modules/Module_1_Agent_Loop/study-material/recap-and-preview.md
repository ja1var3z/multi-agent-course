## Before we start
This is the opening module. No prior lesson to recap — just get oriented.

**You're in the right place if:** you've used ChatGPT or Claude, maybe tried a coding
tool, and want to actually build and understand agents rather than copy-paste from a chatbot.

## Coming up (Module 01 — The Agent Loop: ReAct, Architectures & the Claude Code Harness)
What you'll be able to do after today:
- Explain what makes an LLM an "agent" and why you'd build one
- Trace the agent loop: prompt → intent → tool execution → observation → repeat
- Tell the four levels of agentic architecture apart and pick the right one
- Explain ReAct (reasoning + acting) and what an agent harness adds around the model

**Watch for:** the difference between the *model* (which thinks) and the *harness* (which
does). That distinction is the backbone of everything later.

## If you only remember one thing walking into class
> An agent isn't a smarter chatbot — it's a model running in a loop, using tools, until a
> goal is met. Everything today builds on that loop.
