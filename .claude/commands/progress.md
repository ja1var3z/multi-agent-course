# /progress — Show Where the Learner Stands

Run this when the learner types `/progress` or asks "where am I?", "what's next?", etc.

## Steps

1. Read `progress/learner-progress.md`.
2. Summarize concisely:
   - Modules completed, in progress, not started.
   - Their chosen learning style.
   - Any weak spots flagged from quizzes.
3. Recommend a clear next step (continue current module, review a weak spot, take a quiz).
4. Ask if they'd like to do that now.

Keep it short — a status check, not a lecture.
